# E-commerce Sales Data Analysis Project

## Objective
The main objective of this project is to analyze e-commerce sales data, clean missing and unstructured values, and generate business insights using Python libraries such as pandas, numpy, and matplotlib.

## Workflow
1. Data Import
2. Data Cleaning
3. Feature Engineering
4. Data Visualization
5. Insights and Conclusions

## Tools Used
Python | Pandas | NumPy | Matplotlib | Jupyter Notebook | Excel
